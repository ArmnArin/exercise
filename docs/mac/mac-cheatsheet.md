# üìÇ File & Folder Commands

| Command | Description |
|---------|-------------|
| `ls` | List files |
| `ls -a` | List all files (nondetailed) |
| `ls -la` | List all files (detailed , including hidden without inode number) |
| `ls -lisa` | List all files (detailed , including hidden with inode number) |
|   etc.  |     etc.    |

An inode number is a unique identifier that the Linux/Unix filesystem uses to store information about a file or directory.
